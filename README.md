# Arnold Kuchocha Mafuri — Personal Portfolio Website

A responsive personal portfolio website built with **HTML, CSS, and JavaScript**. It showcases my skills, and projects as an aspiring Cloud Engineer.

🔗 **Live Site:** [arnoldkm278.github.io/Portfolio](https://arnoldkm278.github.io/Portfolio/)

---

## 📸 About The Project

This portfolio website was designed and built from scratch using pure HTML, CSS, and JavaScript — no frameworks or libraries. The goal was to create a clean, professional, and fully responsive site where anyone can learn about who I am, what I can do, and the projects I have worked on.

---

## ✨ Features

- ✅ Fully responsive design — works on desktop, tablet, and mobile
- ✅ Mobile hamburger menu — navigation collapses into a menu on smaller screens
- ✅ Skills section — highlights my technical and soft skills
- ✅ Projects section — displays my hands-on cloud and web projects
- ✅ Contact section — easy way to get in touch
- ✅ Clean and modern dark-themed UI

---

## 🛠️ Built With

| Technology | Usage |

## 🚀 Getting Started

To run this project locally:

```bash
# Clone the repository
git clone https://github.com/arnoldkm278/Portfolio.git

# Navigate into the folder
cd Portfolio

# Open in your browser
Open index.html
```

Or just visit the live site: [arnoldkm278.github.io/Portfolio](https://arnoldkm278.github.io/Portfolio/)
